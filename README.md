# Twitter-Analysis-Dashboard
This project analyzes Twitter engagement data using Microsoft Power BI. An interactive dashboard visualizes metrics like likes, retweets, replies, impressions, and media interactions. Various filters and visualizations are used to explore engagement patterns and understand factors influencing tweet performance.


Multiple visualizations including scatter charts, bar charts, and line charts are used to explore relationships between engagement metrics. Various filters and analytical conditions are applied to identify meaningful trends in tweet interactions.
This project was developed as part of a Data Analytics Internship to demonstrate skills in data visualization, dashboard development, and analytical reporting.

Tools Used
Microsoft Power BI
DAX (Data Analysis Expressions)
Microsoft Excel
